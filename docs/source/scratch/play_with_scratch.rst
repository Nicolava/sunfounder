Scratchで遊ぼう (Pi 5用ではない)
====================================

Scratchは、主に8歳から16歳の子供たちを対象とした教育的なコーディングツールとしてのブロックベースのビジュアルプログラミング言語およびウェブサイトです。サイトのユーザーは、ブロックのようなインターフェースを使用してWeb上でプロジェクトを作成することができます。このサービスはMITメディアラボによって開発され、70以上の言語に翻訳され、世界のほとんどの部分で使用されています。

ここでは、Raspberry PiでScratch 3を使用する方法と、オンラインのScratchでは不可能なScratch 3でRaspberry Pi GPIOにアクセスする方法を学びます。

.. note::

    Scratch 3でプログラムする際、より良い体験のためにはスクリーンが必要です。参考： `Raspberry Piに接続 <https://projects.raspberrypi.org/en/projects/raspberry-pi-setting-up/3>`_ 。もちろん、スクリーンがない場合でも、Raspberry Piのデスクトップにリモートでアクセスすることができます。詳しいチュートリアルは :ref:`remote_desktop` を参照してください。
