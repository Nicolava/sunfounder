速度センサーモジュール
========================

.. image:: img/speed_sensor1.png
    :width: 300
    :align: center

速度センサーは送信機と受信機の2部分から構成されています。送信機から光が出力され、その光は受信機に入ります。

障害物が送信機と受信機の間の光路を遮断すると、受信機は光を検出できず、D0ピンはローレベルを出力します。

.. note::
    このモジュールのA0ピンは空で、回路がありません。
