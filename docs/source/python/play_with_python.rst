 


Jouer avec Python (pas pour Pi 5)
======================================

.. warning::

    ``RPi.GPIO`` est une bibliothèque GPIO largement utilisée, mais malheureusement, elle n'a pas été mise à jour pour prendre en charge le dernier Raspberry Pi 5. Par conséquent, tous les programmes Python écrits en utilisant cette bibliothèque ne seront pas compatibles avec le Pi 5. Il est essentiel de s'adapter aux nouvelles bibliothèques comme ``GPIO.Zero`` pour une compatibilité optimale avec le matériel le plus récent.
