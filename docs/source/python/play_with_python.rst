Pythonで遊ぼう (Pi 5用ではない)
===================================

.. warning::

    ``RPi.GPIO`` は、広く使用されているGPIOライブラリですが、残念ながら、最新のRaspberry Pi 5をサポートするために更新されていません。その結果、このライブラリを使用して書かれたPythonプログラムは、Pi 5と互換性がありません。最新のハードウェアとのシームレスな互換性を確保するために、 ``GPIO.Zero`` のような新しいライブラリに適応することが不可欠です。
