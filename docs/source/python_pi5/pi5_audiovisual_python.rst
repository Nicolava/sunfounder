視聴覚
=============

.. note::

    カメラ モジュールを使用する場合、
    より良いエクスペリエンスを得るために画面が必要になる場合があります。
    `Connect your Raspberry Pi <https://projects.raspberrypi.org/en/projects/raspberry-pi-setting-up/3>`_ を参照してください。 
    もちろん、画面がない場合は、Raspberry Pi デスクトップにリモートでアクセスすることもできます。
    詳細なチュートリアルについては、 :ref:`remote_desktop` を参照してください。



.. When use the camera module, you may need a screen for a better experience, 
.. refer to: `Connect your Raspberry Pi <https://projects.raspberrypi.org/en/projects/raspberry-pi-setting-up/3>`_. Of course, if you don't have a screen, you can also access the Raspberry Pi desktop remotely, for a detailed tutorial please refer to :ref:`remote_desktop`.
